# 示例 Markdown 文档

这是一个用于演示 Code Frontmatter (CFM) 在 Markdown 文件中使用方式的示例文档。

## 项目概述

Code Frontmatter 让每一个代码文件拥有自己的"身份证"，AI 无需读取全文即可理解项目结构。

## 快速开始

1. 安装 MCP Server
2. 在代码文件顶部添加 CFM 表头
3. AI 助手即可自动扫描并利用表头信息
